# Analysis Outputs Methodology

This script generates derived tables from the unified lifecycle dataset.

The analysis intentionally focuses on outputs that are useful now:

- annual lifecycle funnel;
- auction competitiveness;
- property category summaries;
- repeat buyer activity;
- top bid multiples;
- review flags for remaining OCR caveats.

Run:

```powershell
python -m src.analysis.build_analysis_outputs
```
